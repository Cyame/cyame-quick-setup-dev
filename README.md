# cyame-个人开发环境快速搭建
A personally shared dev-env setup plan with some softwares.

自用 公开开源

## 安装基础依赖*（推荐按顺序）*

- Node.js
- Git
- Python 3.7(Recommanded)
- yarn
- VSCode
- MinGW for C/C++

## 运行安装脚本

- Python(PYPI)
```
pip install -r *.txt
```

- Node
```
npm i
```
at local.

Bestly manually add
```
npm i -g *
```
packages written in [Node](node.md)
